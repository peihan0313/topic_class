安裝
    安裝nodejs 8.9.4已上
    開啟cmd 在檔案當前目錄執行npm install --save
    安裝完成

執行
    將要套入的頁面放入此資料夾ex. p1.htm ,p2.htm ,... 
    將線上的網址對應頁面[http://xxxx/xxxxx/p1.htm]寫入search.txt
    將AD帳號密碼寫入search.txt {id:yourADid},{pw:password}
    網址用中括號
    AD用大括號
    開啟cmd到此資料夾執行npm start
    結果詳細內容會寫進result.txt
    有失敗也會顯示在cmd裡
